# L-system Flower Plant Generator

<h2>Getting Started</h2>
<h3>Prerequisites</h3>

    Python 3.x
    Libraries: pygame.

<h3>Installation</h3>

Clone the repository:

    git clone https://github.com/your-username/l-system-flower-generator.git
    cd l-system-flower-generator

Install dependencies:

    pip install pygame

<h3>Usage</h3>

Run the script:

    python generator.py
